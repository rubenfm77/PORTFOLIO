ICU Sepsis Early-Prediction

PyTorch · Drift Detection · Clinical ML
PYTORCH · GRU · MLOPS · DRIFT DETECTION

Predicting sepsis onset — then keeping the model honest in production

End-to-end clinical ML pipeline: a two-layer GRU trained on 12-hour ICU time-series windows (PhysioNet 2019 schema) to flag sepsis risk 1–6 hours early. A full MLOps layer monitors predictions in real time, computes Population Stability Index per feature, and fires automated retrain decisions. The degradation simulation catches sensor-recalibration drift on the very first affected batch — zero false retrains in the clean phase.
